# Hash Table Simulator

A simple visualization tool for Hash Table collision resolution.

I put together this simulator for a class demo to show how Separate Chaining, Linear Probing, and Double Hashing behave under the hood, with real-time logs for every probe and step-by-step playback.

**Key Features:**
🔹 **Interactive Visualization:** Watch step-by-step playback of hashing operations.
🔹 **Collision Strategies:** Compare Separate Chaining, Linear Probing, and Double Hashing side-by-side.
🔹 **Runtime Logs:** A live terminal-style console explaining every probe and decision in real-time.
🔹 **Educational Notes:** Built-in theory cards covering hashing basics and clustering.

**Built with:** React 19, TypeScript, Vite 6, Tailwind CSS 4, and Motion.

Designed to make abstract hashing concepts more tangible through visual execution.
